GEH Nails Designer - Arte & Essência
1. Visão Geral
Este projeto consiste em uma experiência digital imersiva desenvolvida para a marca GEH Nails Designer. A interface foi concebida sob o conceito de Psicodelia Rústica, unindo elementos orgânicos, texturas analógicas e movimentos dinâmicos para transmitir a identidade artística da profissional.
2. Tecnologias Utilizadas
A solução foi construída utilizando tecnologias web nativas para garantir máxima performance e compatibilidade:
HTML5: Estrutura semântica e otimizada para SEO.
CSS3: Estilização avançada com foco em Glassmorphism, animações de sistema e responsividade.
JavaScript (ES6+): Lógica de micro-interações, manipulação de scroll e efeitos visuais dinâmicos.
3. Arquitetura de Arquivos
A estrutura do repositório segue um modelo simplificado para facilitar a manutenção e o carregamento:


/
├── index.html     # Ponto de entrada principal
├── main.css       # Definições de cores, fontes e estrutura base
├── style.css      # Keyframes, animações de entrada e interações
├── app.js         # Lógica de performance e efeitos dinâmicos
├── nail1.jpg...   # Ativos de imagem (Galeria)
└── video1.mp4     # Mídia de introdução cinematográfica


4. Diferenciais de Design e Performance
4.1. Conceito Visual
Psicodelia Rústica: Uso de gradientes sunset (hsl) combinados com uma camada de ruído visual (grain-overlay) que emula a estética de filmes analógicos.
Escultura Digital: Substituição de imagens estáticas por formas orgânicas geradas via CSS (artistic-blob) com animações de pulsação suave.
4.2. Experiência do Usuário (UX)
Reveal Líquido: Transições de seção que utilizam clip-path e blur para criar uma revelação gradual e fluida dos conteúdos.
Cursor Magnético: Implementação de um sistema de cursor customizado para dispositivos desktop, aumentando o engajamento interativo.
Tipografia Dinâmica: Efeito de digitação humana na introdução, sincronizado com o término do vídeo de abertura.
4.3. Otimização Técnica
Lazy Loading: Imagens da galeria são carregadas sob demanda para reduzir o consumo de dados e acelerar o tempo inicial de carregamento.
Paralaxe de Alta Performance: Utilização de requestAnimationFrame e translate3d para garantir que os efeitos de rolagem sejam processados pela GPU.
5. Instruções de Instalação e Uso
Clone este repositório.
Certifique-se de que os arquivos de mídia (video1.mp4 e imagens .jpg) estão localizados na raiz do projeto.
Abra o arquivo index.html em qualquer navegador moderno.
